En el BODY que es el cuerpo encontramos un HEADER que es un encabezado en el cual tenemos las etiquetas: - H1 para el titulo. - H2 para subtítulos. - P para el párrafo. - NAV para un menu de navegación y dentro de esta tendremos: - UL para crear una lista desordenada y dentro de UL tenemos una LI para elementos de la lista y dentro del UL tenemos un A para un enlace. Luego tenemos una etiqueta MAIN donde ira el contenido principal del BODY en el cual encontraremos las etiquetas: - SECTION donde crearemos una sección para los productos y dentro de esta tendremos: - H2 para el subtitulo productos - OL para crear una lista ordenada y dentro de esta LI para elementos de la lista. - SECTION donde crearemos otra sección para los empleados y dentro tendremos: - TABLE para crear una tabla y dentro estarán las etiquetas: - THEAD para la cabecera de los elementos de la tabla que irán en negrita esta llevara TR para las celdas Y TH para la información dentro de estas celdas.
