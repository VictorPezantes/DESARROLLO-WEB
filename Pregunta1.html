-----------------Cuerpo Header Header// Etiqueta de encabezado que agrupo los elemento que compondrán este mismo como H1, NAV, UL,Li,A NAV/ Etiqueta que contiene el menú de navegación de un sitio Web UL// lista desordenada que contiene Etiquetas LI// Etiqueta de lista que asu ves crea los enlace mediante el uso anidado de la etiqueta A // Etiqueta de enlace la cual median el atributo Href, enlaza con su destino ----- Etiqueta Main // Etiqueta que contiene el contenido principal de la página en este caso H1// H2 // P Body// Contenedor del cuerpo de la pagina H1 // etiqueta de encabezado H2// Etiqueta de encabezado usado en sub título segundo nivel P// Etiqueta de párrafo, agregar contenido o texto referencial Seccion// Etiqueta que engloba un contenido en particular, denominado sección, el contenido deber coherente entre sí, en esta caso abarca las siguientes etiquetas Table,Theader,TH,TR,TD,TBODY Table// Etiqueta para la creación de tablas Theader// Etiqueta de que demarca el encabezado de una tabla TH// Etiqueta demarcada que destaca los encabezados de una columna TR// Etiqueta que crea la fila de una tabla TD// TABLE DATA, contiene la información de una tabla TBODY// Contiene la información principal de la tabla SECCION (Contacto) Etiqueta Input// Etiquetas para ingresar información especifica Texarea// Etiqueta de ingreso de información específicamente un texto más extenso, principalmente usado para dejar algún mensaje Button// Etiqueta de botón de acción en este caso de tipo submit, el cual permite el envio de la información FOOTER// Etiqueta de pie de pagina
